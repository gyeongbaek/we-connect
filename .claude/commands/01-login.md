## 1. 로그인 페이지

### 개요

사용자 인증을 위한 간편 로그인 페이지

### 기능 요구사항

| 기능           | 설명                                          | 우선순위 |
| -------------- | --------------------------------------------- | -------- |
| 닉네임 로그인  | 닉네임 입력만으로 간편 로그인                 | P0       |
| Discord 로그인 | Discord OAuth2를 통한 소셜 로그인             | P0       |
| 세션 유지      | 로그인 상태 유지 (브라우저 세션/로컬스토리지) | P1       |

### 화면 구성

- 서비스 로고 및 타이틀
- 닉네임 입력 필드 + 로그인 버튼
- "또는" 구분선
- Discord 로그인 버튼

```plaintext
+---------------------------+
|        [서비스 로고]        |
|        we:connect        |
|                           |
|  [닉네임 입력 필드]       |
|  [닉네임으로 로그인 버튼]   |
|        --- 또는 ---        |
|  [Discord 로그인 버튼]     |
+---------------------------+
```

### API 엔드포인트

```

POST /api/auth/login - 닉네임 로그인
GET /api/auth/discord - Discord OAuth 시작
GET /api/auth/discord/callback - Discord OAuth 콜백
POST /api/auth/logout - 로그아웃

```
