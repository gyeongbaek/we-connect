## 4. 프로젝트 페이지

### 개요

진행 중인 프로젝트 관리 및 빠른 접근을 위한 허브 페이지

### 기능 요구사항

| 기능           | 설명                                  | 우선순위 |
| -------------- | ------------------------------------- | -------- |
| 프로젝트 목록  | 카드 형태로 프로젝트 표시             | P0       |
| 프로젝트 검색  | 이름으로 프로젝트 검색                | P0       |
| 타입 필터      | 부트캠프/강의기획/개발 등 타입별 필터 | P0       |
| 상태 필터      | 진행중/완료/보류 상태별 필터          | P0       |
| 빠른 링크 접근 | Discord, Notion 원클릭 이동           | P0       |
| 프로젝트 추가  | 새 프로젝트 생성                      | P0       |
| 참가 인원 표시 | 프로젝트별 담당자/참여자 표시         | P1       |

### 프로젝트 카드 상세

```
+---------------------------------------+
|  🚀 개발 프로젝트              [⋮]    |  ← 타입 + 더보기 메뉴
|                                       |
|  we:connect                           |  ← 프로젝트명
|                                       |
|  팀 협업 도구 MVP 개발                 |  ← 설명 (1-2줄)
|                                       |
|  👥 5명                               |  ← 참가 인원수
|  (A)(B)(C)(L)(+2)                     |  ← 참가자 아바타
|                                       |
|  +-------------+  +-------------+     |
|  | 💬 Discord  |  | 📝 Notion   |     |  ← 빠른 링크 버튼
|  +-------------+  +-------------+     |
|                                       |
|                          ● 진행중     |  ← 상태 뱃지
+---------------------------------------+
```

### 데이터 모델

```typescript
Project {
  id: string
  name: string
  description: string
  type: ProjectType
  status: ProjectStatus
  discordLink: string | null
  notionLink: string | null
  participants: Participant[]
  createdAt: datetime
  updatedAt: datetime
}

ProjectType =
  | "BOOTCAMP"       // 부트캠프
  | "LECTURE"        // 강의 기획
  | "DEVELOPMENT"    // 개발 프로젝트
  | "CONTENT"        // 콘텐츠
  | "OTHER"          // 기타

ProjectStatus =
  | "IN_PROGRESS"    // 진행중
  | "COMPLETED"      // 완료
  | "ON_HOLD"        // 보류

Participant {
  userId: string
  nickname: string
  role: string
}
```

### API 엔드포인트

```
GET    /api/projects                    - 프로젝트 목록 조회
       ?search={keyword}                - 검색어
       &type={type}                     - 타입 필터
       &status={status}                 - 상태 필터
POST   /api/projects                    - 프로젝트 생성
GET    /api/projects/:id                - 프로젝트 상세
PUT    /api/projects/:id                - 프로젝트 수정
DELETE /api/projects/:id                - 프로젝트 삭제

POST   /api/projects/:id/participants   - 참가자 추가
DELETE /api/projects/:id/participants/:userId  - 참가자 제거
PUT    /api/projects/:id/status         - 상태 변경
```
