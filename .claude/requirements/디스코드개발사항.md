# 디스코드 연동

## 출근하기

- 출근하기를 등록한 후, 수정하기 버튼 오른쪽에 디엠 버튼을 클릭하면 디스코드 특정 서버의 채널로 메시지를 전달합니다.
- 텍스트는 다음과 같이 작성됩니다.

1. 8시간 근무인 경우, 9-18시 근무이면 전송되지 않습니다.
2. 시간이 변경된 경우, 해당 시간 근무로 메시지가 전송됩니다.
   -> "10-19시 근무하겠습니다"
3. 휴가가 등록된 경우 휴가로 메시지가 전송됩니다.
   -> "오늘 휴가입니다"/ "오전 휴가입니다" / "오후 휴가입니다" / "14-16시 휴가 사용하겠습니다."
4. 휴가가 등록되었고, 근무 시간이 조정된 경우
   "오전 휴가 사용하고 14-19시 근무하겠습니다"
5. 추가 근무나 조기 퇴근을 등록한 경우
   "10-19시 근무하겠습니다 (1시간 추가 근무)" / "9-16시 근무하겠습니다 (2시간 조기 퇴근)"
6. 휴가와 추가 근무/조기 퇴근이 함께 등록된 경우
   "오후 휴가 사용하고 10-16시 근무하겠습니다 (1시간 조기 퇴근)"

## 업무일지

디스코드로 전송하기를 누르면 다음과 같은 형식으로 전송됩니다.

```
[${currentUser.name.slice(1)} 일지] @{discordName} `업무startTime-endTime` 근무장소
- 프로젝트
  - 태스크1
  - 태스크2
- 프로젝트2
  - 태스크1
- 단기 업무1
```

- 근무 장소는 출근하기에 등록된 장소가 표시됩니다.
  - 장소는 오전, 오후가 동일한 경우 한 번만 표시됩니다.
  - 장소가 다른 경우 재택이 아닌 경우가 표시됩니다. (오후 아라 사무실 / 오전 오피스제주)
- 업무가 없는 경우 디스코드 전송이 활성화되지 않습니다.

## 서버와 채널, 계정 연동

- 디스코드 계정을 연동할 수 있습니다. 프로필에서 연동합니다.
- 서버와 채널은 공통으로 적용됩니다. 이오 관련해서 필요한 태스크를 정리해주세요.
