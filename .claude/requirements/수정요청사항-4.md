# 프로젝트 페이지

## 검색 및 필터링

- 검색과 필터링 영역을 가로로 배치합니다.

## 프로젝트 상세 페이지

- 탭 영역으로 나뉘어 있고 상단에는 프로젝트 명과 오른쪽에 디스코드 링크, 노션 링크 버튼이 있습니다.
- 폼 형태가 아닌 데이터를 볼 수 있는 형태로 나열해주세요.
- 노션 형태로 마크다운을 추가할 수 있또록 수정해주세요.

## 새 프로젝트

- 새 프로젝트 버튼은 배경색을 추가해주세요
- 프로젝트 명과 외부 링크를 입력받고, 담당자를 등록할 수 있습니다. 담당자를 등록된 경우 업무 일지 프로젝트 선정 시 상단에 배치되고, 담당자가 아니더라도 업무 일지에서 해당 프로젝트를 진행할 수 있으며 진행하면 참여자로 등록됩니다.
- 참여자는 담당자로 추가할 수 있습니다.
- 프로젝트 생성 시 템플릿을 선택할 수 있습니다. (개발 프로젝트, 강의 프로젝트, 기획안) 템플릿 없는 빈 프로젝트도 가능합니다.
