# 근무 기록 페이지

## workHistory

- 주간 근무 기록을 확인할 수 있습니다.
- 이전 기간의 항목을 클릭하면 근무 시간 조정, 휴게 등록, 휴가 등록 모달을 출력하여 근무 기록을 수정할 수 있습니다.
- 재택을 나타내는 아이콘은 제거해줘
- 12월 근무에서 부족으로 뜨는데 전체 기간에서 8시간씩 근무한다고 가정하고, 오늘 날짜를 기준으로 추가 근무를 한 시간이나 덜 근무한 시간을 계산하는 형식으로 변경합니다.
- 전체 내역을 클릭하면 별도의 페이지로 이동합니다. `attendance/month`로 이동하고 월별로 근무 기록을 볼 수 있습니다.

## 휴가현황

- 휴가 현황은 유급휴가, 무급휴가, 창립일, 생일반차를 제외하고는 신청을 해야합니다. 시간 옆에 신청 필요라고 출력됩니다.
- 월의 사용 휴가도 출력됩니다. 오늘의 휴가는 해당 날짜에 색이 입혀집니다.
- 시간은 오른쪽으로 정렬하고 사용 가능 시간/총 시간을 같이 출력합니다.

## 캘린더

- 캘린더 하이라이트의 border, outline을 더욱 연하게 설정해주세요.
- 클릭하면 모달이 생겨 이벤트를 추가할 수 있습니다.

  - 항목 이름 작성 (필수)
  - 날짜 선택 (필수)
  - 기간 설정 가능
  - 특정 시간 설정 가능

  - 참가 인원 표시 가능

- 목 데이터를 추가해주세요.
