# 예약 페이지

## 일간 예약 캘린더

- [ ] 예약 항목을 드래그하여 위치를 변경하거나, 위 아래를 클릭 드래그 하여 범위를 변경할 수 있습니다.
- [ ] 클릭해서 예약 상세 모달이 출력되었을 때 수정이 가능하도록 합니다. [수정하기][닫기]
- [ ] 캘린더 크기를 수정해주세요. 현재 1시간으로 범위를 설정하는 경우 요소의 입력된 값들이 확인이 되지 않습니다. 제목 (시간) 이렇게 표시해주세요.

## 주간 예약 캘린더

- [ ] 사용자 이름이 2글자로 나오는데 전부 출력되도록 합니다.

## 예약 모달

- [ ] 내 예약에 대한 버튼 디자인을 수정해주세요.
- [ ] 제목 / 날짜 + 시간 / 장소 이름(강의실, 방송실1, 방송실2) / 담당자(사용자 이름) 항목이 보이도록 합니다.
- [ ] 메모 항목은 제거해주세요.

## 모달 디자인

- [ ] 전반적인 모달 디자인의 버튼 스타일을 확인해주세요. 예를 들어 예약 페이지의 예약하기 버튼은 배경과 테두리가 없어 버튼의 범위를 파악하기 어렵습니다. 배경을 active한 색으로 변경해주고 일관되도록 디자인을 수정해주세요. 취소는 검정색 테두리라 레이아웃이 동떨어져 보이는데 연한 회색으로 바꿔주세요.
- [ ] 모달을 모달 외부를 누르면 모달이 닫히도록 합니다.
