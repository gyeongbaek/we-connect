# 출근 기록 페이지 레이아웃 수정

## 레이아웃 재배치

```plaintext
+----------+----------+
|   캘린더   |  출근하기  |
|          |----------+
|          |  휴가현황  |
+----------+----------+

```

### 출근하기

```plaintext
+-------------------------+
| 출근하기               8h |
+-------------------------+
| 오전(9-12)    오후(13-18) |
| [드롭다운]     [드롭다운]   |
+-------------------------+
|     [시간 조절 바]        |
+-------------------------+
| 기간 설정(아코디언)          |
| 시작일 [ ] ~ 종료일 [ ]    |
+-------------------------+
|      [출근하기 버튼]      |
+-------------------------+
```

- 등록하기를 누르면 기본적으로 오늘 하루의 근무가 적용됩니다.
- 기간 설정을 한 경우, 자동으로 해당 기간 동안의 출근이 등록됩니다.
  휴가 현황을 드래그해서 휴가를 등록할 수 있습니다.
- 시간 조절 바를 클릭해서 드래그 하여 근무 시간을 세부적으로 조절할 수 있습니다.
  - 점심 시간은 클릭 활성화한 후에 이동 시킬 수 있습니다.
  - 오전/휴게/오후의 경계에서 드래그하여 시간을 조절할 수 있습니다.
