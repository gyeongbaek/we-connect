# 기본 레이아웃

- **사이드바**: 주요 섹션 (대시보드, 출퇴근, 업무관리, 보고서 등) 네비게이션
- **헤더**: 현재 섹션 제목, 사용자 프로필, 알림 아이콘
- **메인 콘텐츠 영역**: 각 섹션별 주요 기능 및 정보 표시

```plaintext
+-----------------------------------------------------+
| we:connect    [닫기 버튼]   | [섹션 제목]               |
| [출근 기록]                  |   [메인 콘텐츠 영역]        |
| [업무 일지]                    |                        |
| [프로젝트]                  |                         |
| [팀 현황]                    |                        |
| ---------                   |                        |
| [내 프로필]                    |                        |
| [로그아웃 버튼]                    |                        |
```
